Lesson 22

Grammar - Kdyby bývalo bylo

🪀 Third Conditional aneb co už dnes bohužel nezměníme

If + past perfect, would have + základní tvar slovesa

Third Conditional se používá k diskusi o minulých situacích, které se nestaly, a jejich hypotetických výsledcích. Jedná se o způsob, jak si představit jiný výsledek, než jaký ve skutečnosti nastal. Tato forma je zvláště užitečná pro přemýšlení o promeškaných příležitostech, představování alternativních scénářů a pochopení důsledků akcí, které nebyly provedeny.

😥 Analyzing missed ppportunities

🧩 If I had studied computer science, I would have pursued a career in programming. = nicméně už je pozdě, univerzitní léta mám za sebou

🧩 If I had attended that coding workshop, I would have improved my programming skills. = workshop již proběhl

🧩 If they had considered user feedback, they would have made a more user-friendly app. = appka byla již spuštěna a díru do světa neudělala

😥 Reflecting on different outcomes

🧩 If we had implemented better security measures, we would have prevented the data breach.

🧩 If she had backed up her files, she wouldn't have lost her important documents.

🧩 If our team had tested the software thoroughly, we would have found the bug before the launch.

🧩 If we had upgraded our system, we wouldn’t have faced so many performance issues.

🧩 If he had remembered to update the antivirus, his computer wouldn’t have been infected with malware.

🧩 If our company had invested in cloud storage, we would have had easier access to our data.
