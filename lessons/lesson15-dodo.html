Lesson 15

Grammar - Možná, můžeš, aby, když už...

Dalšími modálními slovesy, na které se v této lekci zaměříme, jsou may a might. Tato slovesa mají daleko širší využití, než jste si nejspíš mysleli, a dají se použít k vyjádření možností, svolení, účelu i k návrhům. Z gramatického hlediska se chovají stejně jako can a could, takže postrádají obvyklou časovost, a často se dají s can a could zaměnit, aniž by věta dramaticky změnila význam.

💪 Possibility = něco je možné

Chceme vyjádřit, že něco je možné, ale není to jisté. Obecný pravidlem je, že might vyjadřuje menší šanci, že se něco stane, než may, podobně jako could má menší pravděpodobnost než can. V určitých případech may a might mohou kontextově odkazovat na budoucnost. ➰

🧩 There may be a bug in the system. = jen spekulujeme, jistě nevíme

🧩 Our new colleague may come to the party tonight. = Je možné, že přijde, ale není to jisté.

🧩 The server may experience downtime during the maintenance window tonight. = ale možná k tomu nakonec nedojde

🧩 Due to the recent software update, users may notice the app running faster.

🧩 The new patch might fix the security vulnerability we identified last week.

🧩 If we don't address the memory leak issue, the system might crash under heavy load.

🧩 Due to high traffic, the website may load slower during peak hours.

🧩 Users may encounter errors if they don't update their browsers to the latest version.

🧩 The cloud storage might be temporarily unavailable during the migration process.

🧩 The integration of the two platforms might lead to some initial synchronization issues.

🙏 Permission = svolení

Dáváme povolení k činnosti nebo o povolení zdvořile žádáme. V tomto kontextu je možné libovolně zaměnit s "can," aniž by věta změnila význam.

🧩 May I push the newest version to the repository?

🧩 May I go to the restroom?

🧩 You may access the database only after you complete the security training.

🧩 Developers may use the testing environment for their integration tests.

🧩 Team leads may grant temporary permissions to junior developers when necessary.

🧩 Only those with admin privileges may restart the server.

🙏 Purpose = účel

Pomocí "may" je možné šikovně vyjádřit účel nebo specifický způsob, jak něco udělat.

🧩 She's saving money so that she may buy a new laptop.

🧩 We've added extra layers of encryption so that sensitive data may remain protected from breaches.

🧩 We're conducting this code review so that any potential issues may be identified before deployment.

🧩 We've implemented strict password policies so that unauthorized access may be prevented.

🧩 The IT department is deploying patches so that all workstations may have the latest security updates.

🧩 We're introducing the new software in phases so that potential issues may be addressed incrementally.

🙏 Suggestions = návrhy

K vyjádření návrhů se často používá ustálené slovní spojení "might as well," které se dá do češtiny přeložit jako "proč ne, když už, stejně tak, při jednom."

🧩 We might as well go home if the event is canceled.

🧩 Considering the recurring bugs in the current platform, we might consider migrating to a more robust solution.

🧩 Given the rise in cyber threats, we might as well invest in advanced cybersecurity measures this quarter.

🧩 Since we're already updating the user interface, we might as well address those usability issues that the UX team pointed out.

🧩 If we're going to be performing maintenance on the server tomorrow, we might as well upgrade the operating system too.

🧩 Considering the consistent feedback about the app's slow performance, we might as well allocate resources to optimize its speed in the next sprint

❌ Might not/may not

Záporné tvary těchto modálních sloves sice mají své stažené podoby, v současné době se však už téměř nikdy nepoužívají, a proto není třeba se je učit.

🧩 The software update might not be compatible with older operating systems.

🧩 Due to server maintenance, the website may not be accessible during off-peak hours.

🧩 The cloud storage solution might not offer enough space for our growing database needs.

🧩 If the code isn't optimized, the program may not run efficiently on older hardware.

🧩 The mobile application may not be available in all regions due to licensing restrictions.

🤔 May be nebo maybe?

I když tato slovíčka vypadají téměř identicky, a proto je studenti často zaměňují, jde ve skutečnosti o dva odlišné slovní druhy. Modální sloveso "may" ve spojitosti s "be" je modální sloveso následované plnovýznamovým slovesem "být" zatímco "maybe" psané dohromady je příslovce znamenající "možná"

🧩 The unexpected system shutdown may be related to the recent software update.

🧩 The corrupted files may be a result of the malware attack we experienced last week.

🧩 Maybe we should consider migrating to a different cloud provider.

🧩 If the code keeps returning errors, maybe there's an issue with the libraries we're using.

🧩 I received a notification about a system update; maybe that will fix the compatibility issues we've been facing.

🧩 I've heard about a new cybersecurity tool; maybe we can test it out next month.

🧩 The app's UI isn't as intuitive as I hoped; maybe we can get some feedback from the UX team.

🧩 I can't access the shared drive; maybe there are permission settings that need to be adjusted.

Vocabulary - Slovíčka vyjadřující pravděpodobnost

Probability [ˌprɒbəˈbɪlɪti]

The measure of the change that an event will occur, often used in a formal, precise and mathematical context.

🧩 The probability of data loss is minimal with our new backup system.

🧩 Given the system's robustness, the probability of a crash is low.

🧩 The IT department calculated the probability of a cyberattack based on recent trends.

Likelihood [ˈlaɪklihʊd]

The chance or probability of something happening, informal, less precise.

🧩 Given the current vulnerabilities, there's a high likelihood of a security breach.

🧩 The likelihood of a software glitch increases if regular updates are skipped.

🧩 The IT team assessed the likelihood of server downtime during the upgrade.

Chance [ʧɑːns]

A possibility of something happening, informal.

🧩 There's a good chance that the software patch will resolve the current issues.

🧩 The IT team took no chances and implemented multiple security layers.

🧩 What's the chance of recovering the lost data after the system crash?

Possibility [ˌpɒsɪˈbɪlɪti]

A thing that may happen or be the case.

🧩 We're exploring the possibility of migrating to a cloud-based solution.

🧩 There's a slight possibility that users might experience downtime during the update.

🧩 Given the latest advancements, remote work is now a viable possibility for most IT professionals.

Odds [ɒdz]

The likelihood of a particular outcome, often used in the context of gambling but it can be used more broadly.

🧩 The odds of encountering a virus are high if you don't have antivirus software.

🧩 Given the system's age, the odds of hardware failure are increasing.

🧩 With the current security measures, the odds of unauthorized access are minimal.

Certainty [ˈsɜːtənti]

A firm conviction that something is the case; absolute confidence.

🧩 With our new encryption techniques, the certainty of data protection is high.

🧩 Given the test results, there's a certainty that the software is bug-free.

🧩 The IT manager expressed certainty in the team's ability to handle the upcoming challenges.

Listening - What is HTML?

Hypertext Markup Language, or HTML, is a language that defines the structure of content on a webpage. Development started in 1989 at CERN by Tim Berners-Lee, who had just invented the world's first web browser, World Wide Web, the origin of WWW, but he needed a publishing language to display content within it. So, he chose a syntax based on the Standard Generalized Markup Language, which uses opening and closing tags to provide meaning to otherwise unorganized text, like "h" for heading, "p" for paragraph, and most importantly, "a" for anchor to link to a different web page.

The language underwent many changes during the initial rise of the internet and was standardized as HTML5 in 2008, the version we know and love today. Modern HTML contains tags to handle a variety of different media types, like video, audio, and canvas to draw custom graphics on the fly.

The typical HTML element starts with an opening and closing tag. In the middle, we have content which might be text or additional HTML elements called children. The opening tag may also have one or more attributes, which are key-value pairs that can customize the appearance or behavior of the element.

To build your first web page, create a file named index.html. This file contains a hierarchy of nodes based on the Document Object Model, or DOM. A webpage starts with an HTML tag and has exactly two children: a head and a body. The head contains things that aren't directly shown in the webpage, like a title or a meta tag to give the page a description. But notice how the meta element has no closing tag.

Closing tags are generally optional when an element is fully self-contained. The body contains the actual UI for the end user. We might start with an h1 heading, followed by a paragraph of content. To add an image, we use an image tag, which you'll notice is self-closing. Then, define the source attribute as the file path to the image, and you'll want to give it an alt description to make it accessible to robots and screen readers.

What makes web pages revolutionary, though, is their ability to link to other web pages on the information super highway. Use an anchor tag to create a link, then define the href attributes as the webpage that you want to navigate to when clicked.

HTML also provides structural tags like division, or div, which are often used to group elements together where they can be styled to represent complex UI elements like navbars. When you're happy with your code, grab the full file path to the HTML file and paste it into any web browser. Congratulations, you just built your first website.

Writing - Možná, nejspíš, můžu

📝 Napište 10 vět v jakýchkoli kombinacích významů s použitím may, might (not) a příslovce maybe.

🧩 Maybe I should buy a new desktop, my current one is slow.

🧩 I may go to the office or I may work remote.
