Lesson 31

Grammar - Počitatelnost a nepočitatelnost kontextově

Teď si to trochu ztížíme a zvýšíme level. 😎 I když je k pochopení počitatelnosti a nepočitelnosti na začátku užitečné mít před sebou jasná pravidla a výčty nepočitatelných podstatných jmen k tomu mohou pomoci, ve skutečnosti u některých podstatných jmen (a není jich málo) koncept počitatelnosti a nepočitatelnosti záleží na kontextu, ve kterém je dané slovo použito.

Abstraktnost / obecnost / substance = nepočitatelné

Konkrétnost / substance přenesená do jednotlivosti = počitatelné

💡 Light

Countable: We use LED lights to indicate the status of our servers.

Uncountable: Fiber optic cables use light to transmit data over long distances.

📄 Paper

Countable: She has authored several papers on cybersecurity. = odbornému akademickému článku se často říká "paper"

Uncountable: Paper is made of wood.

👨‍💼 Work

Countable: His works on software development have been widely recognized. = výstupy, články

Uncountable: Work is hard sometimes.

👨‍💻 Experience

Countable: He has had diverse experiences with different programming languages.

Uncountable: Hands-on experience is crucial in IT.

🏢 Room

Countable: Our new office has several rooms dedicated to server hardware. = pokoj

Uncountable: We need to ensure there's enough room for cooling systems in the data center. = prostor

🥛 Glass

Countable: I have two favourite glasses I use to drink juice. = sklenice jako nádobí

Uncountable: The use of glass in fiber optics has revolutionized telecommunications. = sklo jako materiál

Countable + always plural: Where are my glasses? I need to clean my glasses because they're dirty. She lost her glasses and can't read anything without them. = brýle 👓

👧 Hair

Countable: We found a single hair that caused a short circuit in the motherboard. = jeden vlas, chlup

Uncountable: Our UX designer has very long hair. = vlasy obecně

⏰ Time

Countable: Did you have a good time at the Python conference?

Uncountable: Time synchronization is critical in networked systems.

☕ Coffee

Countable: During late-night coding sessions, I consume several coffees to stay alert. = v podstatě cups of coffee, ale zkráceně

Uncountable: Coffee is super important in our office, especially during product launches.

🔊 Sound

Countable: The app generates sounds to notify users of different events. = konkrétní zvuky

Uncountable: Good sound insulation is important in server rooms to mitigate noise. = hluk obecně

Countable and uncountable synonyms

Většina nepočitatelných podstatných jmen však zůstává nepočitatelná bez ohledu na kontext, a tak musíme najít synonymum, pokud chceme vypíchnout konkrétnost nebo potřebujeme krapet jiný význam.

✨ Advice (Uncountable) - Tip (Countable) - Suggestion (Countable)

She gave me some valuable advice.

She appreciated the advice given by her mentor.

He shared a useful tip for the software installation.

He made a helpful suggestion for improving the code efficiency.

👔 Work (Uncountable) - Job (Countable)

Work seems endless sometimes.

She just got a new job in the tech industry.

🚝 Travel (Uncountable) - Trip (Countable)

Travel broadens the mind.

Our recent business trips were very productive.

📚 Information (Uncountable) - Knowledge (Uncountable) - Fact (Countable)

We need more information to solve this issue.

His knowledge of programming languages is impressive.

He presented interesting facts during the meeting.

She learned an interesting fact about cloud computing.

🪑 Furniture (Uncountable) - Chair, wardrobe, table... (Countable)

They have beautiful furniture in their office.

Each employee chose a new ergonomic chair for their desk.

📲 Equipment (Uncountable) - Device (Countable)

The laboratory has modern equipment.

Each employee is issued a mobile device for work.

👜 Luggage (Uncountable) - Suitcase (Countable)

Please ensure your luggage is not left unattended.

Her suitcases were lost during transit.

🧪 Research (Uncountable) - Study (Countable)

His research on data encryption is groundbreaking.

They published a study on network security vulnerabilities.

⛺ View (Countable) - Scenery (Uncountable)

What a spectacular view!

What spectacular scenery!

Vocabulary - Nepočitatelná IT podstatná jména

✨ Data /ˈdeɪ.tə/

Facts and statistics collected together for reference or analysis.

🧩 The research team collected extensive data on consumer behavior.

🧩 We have tons of data to process.

✨ Software /ˈsɒf.twɛər/

Programs used by a computer.

🧩 The latest software includes several new features.

🧩 They are developing software for data management.

✨ Hardware /ˈhɑːd.wɛər/

The physical components of a computer.

🧩 The company invested in new hardware to improve performance.

🧩 Hardware is very expensive these days.

✨ Bandwidth /ˈbændwɪdθ/ šířka pásma

The maximum rate of data transfer across a given path.

🧩 The company upgraded its network to increase bandwidth and improve internet speed.

🧩 Streaming high-definition videos requires a significant amount of bandwidth. = kvantifikátor se dá počítat a mít tedy člen

✨ Traffic /ˈtræfɪk/ - provoz

The amount of data moving across a network at any given time.

🧩 Network administrators monitor traffic to ensure optimal performance.

🧩 High traffic can lead to slower internet speeds.

✨ Access /ˈækses/ - přístup

🧩 Employees have remote access to the company's network.

🧩 Access to sensitive data is restricted to authorized personnel only.

✨ Infrastructure /ˈɪnfrəˌstrʌktʃər/

The fundamental facilities and systems serving a country, city, or area, including the services and facilities necessary for its economy to function, often applied to IT as the physical and organizational structures needed for technology operations.

🧩 The new data center will expand the company's IT infrastructure.

🧩 Investing in robust IT infrastructure is crucial for tech startups.

✨ Documentation /ˌdɒkjʊmɛnˈteɪʃən/ dokumentace

Material that provides official information or evidence or that serves as a record, especially used in the context of software or hardware manuals, guides, and release notes.

🧩 Good documentation is essential for software development and maintenance.

🧩 Programmers often rely on documentation for troubleshooting.

✨ Artificial Intelligence (AI) /ˌɑː.tɪˈfɪʃ.əl ɪnˈtel.ɪ.dʒəns/

The simulation of human intelligence in machines that are programmed to think like humans and mimic their actions.

🧩 Artificial intelligence is transforming industries with smart automation.

🧩 Computer scientists are researching ethical concerns related to artificial intelligence.

Listening - What is JavaScript

🔥 Study skills tip 🔥

Victoria isn't a native English speaker. The aim of this listening exercise is get used to different accents. Don't forget that most of your colleagues won't come from Britain or the USA.

Hello, my name is Victoria, and today we are going to talk about what is JavaScript. Let's go. So, what is JavaScript? It is one of the most popular, lightweight programming languages or, in other words, a scripting language that web developers use when developing web pages, applications, servers, or games. It allows you to implement dynamic features on web pages that cannot be done only with HTML and CSS. Many browsers use JavaScript as a scripting language for doing interactive things on the web.

Every time you see animated graphics, elements, colors, slideshows, or any other dynamic features on a website, you are seeing the effects of JavaScript. It improves the user experience by converting a website from a static page into an interactive one. When most people begin learning to code, they usually start with HTML and CSS, then usually move on to JavaScript because these three parts work together to form the foundation of web development. The core of your page is HTML, which contains all of the headers, text and any images you wish to include. CSS controls how the page looks, such as modifying fonts and background colors, and JavaScript is what makes your website come to life.

Many of you may have heard of Java and wondered if it was the same as JavaScript. Actually, no. They're often mixed, and the only thing they have in common is the name Java. Java is typically used for all server-side development. It creates applications that run in a virtual machine or browser, while JavaScript is reserved for developing client-side scripts for functions like interactivity and is run on the browser only. While JavaScript mainly serves web-based programs, its programming features have other uses in many fields.

For example, in web and mobile apps, you can use JavaScript frameworks to create online and mobile apps. JavaScript frameworks are collections of JavaScript code libraries that give developers pre-written code to use for programming functions and tasks. The most popular JavaScript front-end frameworks are React, React Native, Angular and Vue. For example, front-end JavaScript frameworks such as jQuery and React improve design efficiency. They let developers reuse and modify code components without affecting their functionality or value.

Aside from websites and apps, developers can use JavaScript to create simple web servers and back-end infrastructure using Node.js, saving you time and effort in web server building. The built-in HTTP model allows you to create a simple HTTP or server that displays plain text. To handle HTTP requests, you can use Node.js on a web server, Node OS, or a third-party one such as Microsoft Internet Information Services and Apache.

You can also use JavaScript to create browser games. Creating games is a great way to start practicing your JavaScript skills. JavaScript can also be used to create 2D and 3D video games that run in web browsers. As the browsers get more powerful, web-based video games evolve beyond simple platforms. JavaScript frameworks like Phaser.js allow you to quickly and easily develop more advanced games directly from your browser. When used with HTML5, an application programming interface such as WebGL, JavaScript may help in the creation of the game.

There are several JavaScript-based game engines available, including Phaser, GDevelop and KiwiJS which may be used to create graphics, recycle code, and create cross-platform programs. Some most popular games made with JavaScript game engines are Angry Birds, The Wizard and 2048. And here we are. Now you have a better understanding of what JavaScript is.
